## Description
Hexlet Chat is similar to Slack chat (very simplified version)

## Requirements
The minimum version of node.js - 13.2.0

## Dependencies
- axios
- bootstrap
- classnames
- formik
- i18next
- leo-profanity
- react
- rollbar
- socket.io-clien
- web-vitals
- yup

## devDependencies:
- eslint

## Install

### Clone repository
```
git clone git@github.com:DMotorina/frontend-project-12.git
```

### Install
```
make install
```

### Start
```
make start
```
